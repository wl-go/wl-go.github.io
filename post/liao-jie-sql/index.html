数据库基础

数据库 (database)

一个以某种有组织的方式存储数据的集合
保存有组织的数据的容器

表 (table)

一种结构化的文件，可以用来存储特定类型的数据

模式 (schema)

可以用来描述数据库中特定的表信息，也可以用来描述整个数据库（和其中表的关系）

列、数据类型

表中的一个字段，称为列，所有表都是由一个或多个列组成

  • 将数据合理的分成多个列，有利于对数据进行分类、过滤

数据类型，所允许的数据类型，每个表列都有相应的数据类型，它限制或允许给列中存储的数据

  • 数据类型可以帮助正确的分类数据，优化磁盘

行 (row)

表中的一条记录

主键 (primary key)

表中每一行都应该有一列(或几列)可以唯一标识自己。

什么是SQL

Structured Query Language 结构化查询语言，提供一种从数据库中读写 数据的简单有效的方法
