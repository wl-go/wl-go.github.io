Android添加忽略文件(SVN)

可以在.idea/workspace.xml中<component name="ChangeListManager">节点下添加一下内容


<ignored mask="*.iml" />

<ignored path="local.properties" />

<ignored path=".gradle/" />

<ignored path=".idea/" />

<ignored path="app/build/" />

<ignored path="build/" />

<ignored path="其他模块名/build/" />    

<ignored path="app/release/" />

获取Android Key的sha1

keytool -list -v -keystore C:\xxx.jks

获取https证书字符串

keytool -printcert -rfc -file c:\xxx.cer

Linux 查端口占用、杀进程

lsof -i tcp:8080  

kill -9 PID

反编译APK

apktool d xxx.apk
apktool d -s xxx.apk (只反编译资源)

#生成apk
apktool b XXX 

#生成签名key
keytool -genkey -alias demo.keystore -keyalg RSA -validity 40000 -keystore demo.keystore 

#重新打包
jarsigner -verbose -keystore demo.keystore -storepass 123456  -signedjar new.apk -digestalg SHA1 -sigalg MD5withRSA old.apk demo.keystore

Curl 使用(GET、POST)

# GET
curl -v 192.168.0.1:8080/xxx
	
# POST
curl -v 192.168.0.1:8080/xxx -d 'x1=1&x2=2'

防止EditText弹出键盘后顶起布局

  1. 方法一，在oncreate中加入
getWindow().setSoftInputMode(WindowManager.LayoutParams.SOFT_INPUT_ADJUST_PAN);
  1. 方法二，在AndroidManifest.xml中加入
 android:windowSoftInputMode="adjustPan|stateAlwaysHidden"

Android 浏览器SQLite查看工具

 debugCompile ‘com.amitshekhar.android:debug-db:1.0.0’

Mac 安装OpenJDK

brew tap AdoptOpenJDK/openjdk
brew install <version> 
<version> 可以选择adoptopenjdk-openjdk8、adoptopenjdk-openjdk9、adoptopenjdk-openjdk10
